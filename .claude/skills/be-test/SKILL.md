# BE TEST
작업한 backend 코드들에 대해, 테스트 코드를 작성합니다.

## 세부내용
- 테스트 코드는 `src/test/backend/` 아래에 작성합니다.
  - 필요하다면 디렉토리를 새로 만들어 진행합니다.
- 테스트 코드를 작성하고, test를 진행합니다.
  - 테스트 결과는 `/.claude/docs/be-test.md`에 작성합니다.
  - 각 테스트에 feature 넘버를 작성합니다.
  - 테스트 결과를 []에 체크박스로 표현합니다.
